## 💡 よく引っかかるパターン

- **「〜してはならない」の絶対禁止に見える罠** → 実は例外がある（Q8 軌道敷など）
- **「〜のときに限り可能」の条件付きに見える罠** → 実は条件なく禁止（Q9 安全地帯など）
- **標識と標示の違い** → 標識＝立て看板、標示＝路面の線
- **警察官の手信号** → 「正面に平行＝横から来る車」という言い回しに慣れること
